Personal Finance Tracker

A modern and responsive Personal Finance Tracker Web Application inspired by professional fintech platforms like Mint. This application helps users manage expenses, track monthly budgets, visualize spending analytics, and monitor financial activity with a clean and interactive dashboard UI.
